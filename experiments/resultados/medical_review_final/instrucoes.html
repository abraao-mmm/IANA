Instruções para revisão clínica

Contexto

Este conjunto de 30 notas será o gold test set usado para avaliar 4 modelos compactos de extração de entidades nomeadas (NER) e sumarização SOAP. As notas foram extraídas do MIMIC-IV (prontuários reais de UTI em inglês) e já tiveram a extração feita automaticamente por um modelo maior (Qwen 122B). Sua tarefa é validar essa extração.

O que avaliar

Para cada categoria de NER (6) e cada campo de SOAP (6), atribua um dos 3 status:

  • Correto — extração faz sentido clínico e cobre o que está no texto.
  • Parcial — captura parte, mas há omissões importantes ou inclusões inadequadas.
  • Incorreto — erro grave: categoria errada, alucinação, contradição com o texto.

Comentário é opcional — só preencha se quiser detalhar uma correção específica. Médicos com tempo limitado podem só clicar nos status.

Funcionamento

  • Todas as suas avaliações são salvas automaticamente no navegador (localStorage). Pode fechar e voltar depois sem perder.
  • Cada nota tem botão Exportar JSON ao final (opcional).
  • Quando terminar todas as 30, volte ao índice e clique Exportar TUDO (JSON) — baixa um arquivo consolidado com todas as suas avaliações.
  • Envie esse JSON de volta para a equipe.

Aviso sobre caso patológico

Uma das notas (paciente 27306123) é um caso de erro de codificação: o CID indica neurossífilis, mas o texto da nota não menciona sífilis em momento algum. O pipeline deve ter retornado listas vazias ou apenas as condições reais (pé de Charcot, neuropatia). Se viu sífilis sendo extraída, é alucinação.
