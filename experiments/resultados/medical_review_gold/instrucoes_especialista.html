Instruções para Revisão — Gold Test Set

Propósito: Esta é a validação final do test set gold do Projeto IANA. Os resultados desta revisão serão usados para:
  • Avaliar os 5 modelos compactos treinados no silver standard
  • Calcular métricas finais (precision, recall, F1) para o paper na MDPI Diagnostics
  • Documentar a qualidade do silver standard gerado pelo Qwen 3.5-122B

Contexto

Estas 20 notas novas complementam as 10 notas já revisadas anteriormente, totalizando 30 notas no gold test set (10 HIV + 10 TB + 10 Sífilis). As 10 notas anteriores não precisam de nova revisão — estão ancoradas no gold set com as anotações já realizadas.

O que revisar em cada nota

Cada arquivo HTML contém 4 seções:

  1. Nota Original (inglês) — texto bruto do MIMIC-IV
  2. NER Extraído (português) — 6 categorias de entidades
  3. SOAP Estruturado (português) — 6 campos
  4. Tabela de Anotação — para registrar erros encontrados

Tipos de erro a anotar

TipoDescriçãoExemplo
OmissãoEntidade presente no texto original mas não extraída Hipertensão no PMH não aparece em disease_or_syndrome
Categoria erradaEntidade extraída na categoria incorreta "Leucocitose" em disease em vez de lab
Negação vazadaTeste negativo/pendente gerando entidade positiva "RPR-Negativo" gerando "Sífilis" em disease
TraduçãoErro de tradução EN→PT ou termo em inglês residual "Rash" em vez de "Exantema"
InvençãoEntidade que não existe no texto original Sintoma inventado a partir do diagnóstico
OutroQualquer outro tipo de erro—
